Funções

Ao longo de todos os códigos JS desse site foram usadas as famigeradas Funções
elas são como se fossem uns mini programas dentro do seu programa principal, você pode chamar elas quantas vezes quiser e elas podem ser usadas em qualquer lugar do seu código. Elas são criadas com a palavra reservada function e tem o seguinte formato:

function nomeDaFuncao(parametros...){
    //codigo
}

Funções também podem ser declaradas sem ter um nome. E inclusive serem puxadas para variáveis ou para dentro de objetos, arrays e outras funções.

Eventos

eventos são usados para interagir com o usuário. Eles são usados para detectar quando um usuário interage com o site, como clicar em um botão, tocar em certas areas da tela, precionar teclas do teclado ou digitar em um campo de texto.

eles podem ser inline ou declarados pegando o elemento e trazendo-o para o arquivo de script.
Para inline temos um atributo chamado onclick que recebe uma função ou snippet de código em JS. a sintaxe deles quase sempre se baseia em usar o prefixo "on" no inicio do evento:

  • onclick
  • onload
  • ondblclick
  • oncontextmenu
  • onfocus
  • onblur
  • onscroll
  • onresize
  • onkeydown
  • onkeyup
  • onkeypress
  • onmousedown
  • onmouseup
  • onmousemove
  • onmouseover
  • onmouseout
  • onchange
  • onsubmit
    • ...
Mas não são só esses que existem temos eventos de mouse, teclado, formulário, touch, resize, e até eventos para saber se existe um gamepad conectado.
para arquivos separados pegamos o elemento por id ou classe e adicionamos um escutador de eventos. você pode adicionar multiplos eventos no mesmo elemento. Aqui o prefixo "on" não é necessário.

Arrays

Um array é um objeto que pode armazenar multiplos valores. Para criar um array você pode usar o construtor Array(), o simbolo [] ou simplesmente declarar um objeto com chaves numéricas. O array é um objeto especial que possui métodos para manipular seus elementos.

    1        const arr = [
    2            {nome: 'João', idade: 20},
    3            {nome: 'Maria', idade: 25},
    4            {nome: 'Pedro', idade: 30}
    5        ];

Para acessar um elemento do array você pode usar a notação de colchetes [n] passando o índice do elemento que você deseja acessar. O índice começa em 0.

Estrutras de repetição

As estruturas de repetição são usadas para repetir um bloco de código várias vezes, elas são usadas quando você quer fazer algo várias vezes. Isso é bastante importante, pois permite que a repetição de código seja automática, além de percorrer dados de estruturas mais... robustas, se pudermos dizer; elas podem ser criadas com multiplas palavras reservadas diferentes: for while e do while e cada uma tem um uso diferente (apesar de fazerem quase a mesma coisa)

classes

Dez anos atrás chegavam as classes no javascript, e para a alegria de um e a tristeza de outros elas funcionam e em tese tentariam fazer com que você, o programador conseguisse fazer orientação a objetos. Caso você seja alguém as astuto ou sei lá o que, você já deve ter notado que em varias linguagens existem classes e que isso talvez seja foreshadowing para o manifesto ágil. E você estaria correto.

Mas afinal, sr bonanza, o que são as classes?

Classes são as plantas dos objetos, é como se fosse um molde de objeto que você daria valores para ele e ele seria criado para ser usado como quisesse. Antigamente isso era feito com algumas técnicas estranhas, como retornar um objeto por uma função ou copiar ele e modificar os valores.

assim como muitas coisas no javascript classes tendem a ser coisas que a principio você não entende, até mesmo programadores exeprientes não entendem bem o seu uso, visto que isso foi lançado 10 anos atrás o que meio que faz as pessoas experientes ficarem mais acostumadas aos metodos antigos.

você pode fazer mais que ancoras e menus mais avançados. podemos salvar informações no localStorage, guardar como cookies e até mesmo criar um jogo.
